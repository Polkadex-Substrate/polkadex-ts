# Acknowledgements

A huge thank you to the authors and contributors of the following libraries/frameworks:

**React** React makes it painless to create interactive UIs. Design simple views for each state in your application, and React will efficiently update and render just the right components when your data changes. 

**Polkadot.js** Secure cross-chain messaging is here. Connect with more blockchains easily with Polkadot. Unlock the potential of united blockchains. Join an interoperable Web3 with Polkadot.

**Semi Design** Maintained by the Douyin front-end and UED teams, an easy-to-customize modern design system that helps designers and developers create high-quality products.

**Graphql** 
GraphQL is a query language and server-side runtime for application programming interfaces (APIs) that prioritizes giving clients exactly the data they request and no more. GraphQL is designed to make APIs fast, flexible, and developer-friendly.

**Formik** Formik is third party React form library. It provides basic form programming and validation. It is based on controlled component and greatly reduces the time to do form programming. Let us recreate the expense form using Formik library.
